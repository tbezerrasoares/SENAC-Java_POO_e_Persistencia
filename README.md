# 🚀 Desenvolvimento Java: POO, JavaFX e Banco de Dados

Repositório destinado ao armazenamento de atividades, exercícios e projetos desenvolvidos durante a disciplina de **Sistema com Integração a Banco de Dados e Conceitos de POO**.

## 📝 Sobre a Disciplina
A disciplina foca na construção de aplicações completas utilizando a linguagem Java, cobrindo desde os pilares da Programação Orientada a Objetos (POO) até a persistência de dados real com JDBC e criação de interfaces gráficas modernas com JavaFX.

* **Carga Horária:** 60 Horas
* **Linguagem:** Java (JDK 17+)
* **Interface Gráfica:** JavaFX
* **Persistência:** JDBC + MySQL/PostgreSQL

---

## 📅 Cronograma de Estudos

### 🔹 Módulo 1: Fundamentos de POO
- [ ] **Semana 01:** Introdução ao Java e Ambiente (Instalação e Sintaxe).
- [ ] **Semana 02:** Classes e Objetos (Atributos, Métodos e Instanciação).
- [ ] **Semana 03:** Encapsulamento (Modificadores de acesso e Getters/Setters).

### 🔹 Módulo 2: POO Avançada
- [ ] **Semana 04:** Herança (Reutilização de código e palavra-chave `super`).
- [ ] **Semana 05:** Polimorfismo (Sobrescrita e flexibilidade de tipos).
- [ ] **Semana 06:** Classes Abstratas e Interfaces (Contratos de comportamento).
- [ ] **Semana 07:** Membros Estáticos e constantes.

### 🔹 Módulo 3: Interface Gráfica (GUI)
- [ ] **Semana 08:** Introdução ao JavaFX e Ciclo de Vida.
- [ ] **Semana 09:** Layouts (VBox, HBox, GridPane) e Controles de UI.
- [ ] **Semana 10:** Tratamento de Eventos e Scene Builder.

### 🔹 Módulo 4: Persistência de Dados
- [ ] **Semana 11:** Modelagem de Dados e SQL.
- [ ] **Semana 12:** Conexão JDBC e Exception Handling.
- [ ] **Semana 13:** Padrão DAO (Data Access Object) e CRUD completo.

### 🔹 Módulo 5: Projeto Integrador
- [ ] **Semana 14:** Integração Sistema-Banco-Tela.
- [ ] **Semana 15:** Entrega do Projeto Final.
